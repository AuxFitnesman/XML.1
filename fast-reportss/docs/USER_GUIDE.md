# Руководство пользователя

## Начало работы

1. Запустите backend и frontend (см. README.md).
2. Откройте http://localhost:5173.
3. При необходимости нажмите **Войти** или зарегистрируйтесь (личный кабинет).
4. Создайте листовку в редакторе или выберите шаблон.

## Учётная запись и личный кабинет

### Регистрация и вход
- Кнопка **Войти** в панели инструментов → вкладки «Вход» / «Регистрация».
- Пароль — не менее 6 символов.

### Мои проекты
- После входа: **Проекты** — список листовок на сервере.
- **+ Новый проект** — создать пустой макет в облаке.
- **Открыть** — загрузить проект на холст.
- **Удалить** — удалить проект из системы.
- Клик по проекту показывает **историю сохранений** (до 8 последних записей).

### Сохранение на сервер
- **Сохранить** — записать текущий макет (создаёт проект или обновляет открытый).
- Поле названия слева от кнопок — имя проекта.
- Локальный **XML** по-прежнему скачивает файл на компьютер без входа.

### Шаблоны и доступ
- Предустановленные шаблоны (`presets/`) доступны всем.
- Свои шаблоны на сервере видны только вам (через API `/templates`).

## Создание листовки

### Выбор размера
- Стандартные форматы: A4, A5, A3, Letter — в панели свойств справа.
- Пользовательский размер: выберите «Пользовательский» и укажите ширину/высоту в пикселях.
- Ориентация: книжная или альбомная.

### Добавление элементов
- **+ Текст** — текстовый блок. Двойной клик по тексту на холсте открывает быстрое редактирование.
- **+ Блок** — прямоугольник (фон, рамка, заливка).
- **+ Изображение** — загрузка с компьютера. URL можно указать в панели свойств.

### Редактирование на холсте
- Перетаскивание мышью — изменение позиции.
- Выделение + угловые маркеры — изменение размера (для блоков и изображений).
- Панель свойств — точные координаты, шрифт, цвет, выравнивание, жирность, курсив.

### Сетка и слои
- Кнопка **Сетка** — визуальная сетка 20×20 px для выравнивания.
- Панель **Слои** внизу — порядок элементов (z-index). Клик выбирает элемент.

### Отмена и повтор
- Кнопки **↶ Отмена** и **↷ Повтор** — история до 50 шагов.

## ИИ-генерация

1. Запустите backend и frontend.
2. В панели инструментов нажмите **ИИ**.
3. Введите тему (например: «открытие фитнес-клуба, скидка на абонемент»).
4. Выберите тип листовки и режим:
   - **Текст и оформление** — полный макет;
   - **Только текст** — акцент на заголовках и описаниях;
   - **Только оформление** — цвета и композиция (демо-тексты-заглушки).
5. Нажмите **Сгенерировать** — макет появится на холсте. Доработайте вручную и сохраните XML.

Для работы с OpenAI добавьте ключ в `backend/.env` (см. `backend/.env.example`).

## Шаблоны

На главной странице доступны категории:
- реклама, приглашения, объявления, информация, мероприятия.

Выберите шаблон — откроется редактор с готовым макетом. Адаптируйте текст и оформление, затем сохраните.

## Сохранение

**💾 Сохранить XML** — записывает проект во внутренний XML-формат на сервере. Файл хранится в `backend/data/projects/`. Каждое сохранение добавляется в историю.

## Экспорт

1. Выберите формат в выпадающем списке: PNG, JPEG, PDF, SVG.
2. Нажмите **Экспорт** — файл скачается в браузер.

| Формат | Назначение |
|--------|------------|
| PNG | Растровое изображение, без потерь |
| JPEG | Сжатое изображение |
| PDF | Печать и передача документа |
| SVG | Векторная версия макета |

Экспортированные файлы также сохраняются на сервере в `backend/data/exports/`.

## Мои проекты

Список сохранённых листовок на главной странице. Клик — открыть для редактирования. **Удалить** — удалить проект из системы.

## Советы

- Меняйте фон страницы в панели свойств (цвет).
- Для изображений по URL используйте поле в свойствах выбранного элемента.
- Сохраняйте проект перед экспортом, чтобы не потерять изменения.
